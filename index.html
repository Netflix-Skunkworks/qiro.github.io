The Qiro project

Qiro is an agnostic communication library supporting multiple interaction models (fire-and-forget, request-response, request-stream, subscription, channel).

The project is in active development, and the interface is subject to be changed at any time.

Motivation

Composing RPC and stream of data over a distributed system is very hard. Qiro provides powerful abstractions that allow the users to focus on the business logic.

Qiro will have multiple implementation for different underlying protocol. So far we target the reactive-socket, but other protocols maybe added in the future (HTTP, MySQL, Thrift...).
